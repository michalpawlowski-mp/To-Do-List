# To-Do List App 📋

Prosta, responsywna aplikacja do zarządzania zadaniami, stworzona przy użyciu czystego HTML, CSS i JavaScript.

### Funkcjonalności

- **Dodawanie zadań:** Szybko dodawaj nowe zadania do swojej listy.
- **Oznaczanie jako ukończone:** Zaznaczaj zadania, które już wykonałeś.
- **Edycja i usuwanie:** Zmieniaj treść zadań lub usuwaj je z listy.
- **Zapisywanie w przeglądarce:** Aplikacja zapamiętuje Twoje zadania, nawet po odświeżeniu strony.

### Technologie

- HTML5
- CSS3
- JavaScript (ES6+)

---

### Jak uruchomić projekt

1.  Sklonuj repozytorium na swój komputer.
2.  Otwórz plik `index.html` w swojej przeglądarce.
